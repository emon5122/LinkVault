LinkVault

Save everything. Find anything.

LinkVault is a private, offline-first bookmark manager for Android. Save links from anywhere, organize them with folders and tags, add Markdown notes, and find anything instantly — with no account and no tracking.

  • Automatic previews (title, description, image, favicon)
  • Folders, tags, favorites, read later, and pins
  • Instant, highlighted search
  • Import / export (CSV, JSON, HTML bookmarks, Markdown) and local backup
  • Light, dark, and system themes
